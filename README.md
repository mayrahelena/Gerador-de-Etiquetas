# 🏷️ Gerador de Etiquetas

Aplicação desenvolvida em **React com Vite** para geração de etiquetas de produtos em uma mercearia.  
Permite inserir dados dos produtos e gerar etiquetas otimizadas para impressão, com código de barras.

## 🛠️ Como rodar localmente

### ✅ Pré-requisitos

- [Node.js](https://nodejs.org/) (v18 ou superior)
- [Git](https://git-scm.com/) instalado

### 📦 Passos para executar

```bash
# Clone o repositório
git clone https://github.com/mayrahelena/Gerador-de-Etiquetas.git

# Acesse a pasta do projeto
cd Gerador-de-Etiquetas

# Instale as dependências
npm install

# Inicie a aplicação
npm run dev
```

> A aplicação estará disponível em: [http://localhost:5173](http://localhost:5173)

---

## 🧱 Build para produção

```bash
npm run build
```

> A versão otimizada será gerada na pasta `dist/`.

---

## ☁️ Deploy na Vercel

### Configuração recomendada:

- **Framework Preset:** Vite  
- **Build Command:** `npm run build`  
- **Output Directory:** `dist`  
- **Root Directory:** *(em branco, pois o `index.html` está na raiz)*

Você pode conectar diretamente seu repositório do GitHub à [Vercel](https://vercel.com/) para deploy automático.
